## Семинар 1 для 10-11 классов

Первая тетрадка посвящена самому первому знакомству с моделью через GigaChat API.

Что в ней есть:

- настройка доступа к API;
- получение `ACCESS_TOKEN`;
- первый запрос через `curl`;
- тот же запрос через Python;
- удобная функция `ask_model` поверх `requests`;
- эксперименты с промптами;
- генерация промптов из списка;
- структурированный ответ модели;
- простая текстовая история диалога;
- мини-чат;
- модель как классификатор;
- строгий формат ответа;
- вывод о том, что один вызов LLM ещё не делает агента.

На чём акцент:

- сначала запустить код;
- потом понять, что именно произошло;
- потом постепенно усложнять запросы и формат общения с моделью.

Задание в конце семинара:

- попробовать свои промпты;
- поэкспериментировать с форматом ответа;
- посмотреть, как модель меняет поведение от небольших изменений в инструкции.

## Семинар 2 для 10-11 классов

Вторая тетрадка показывает путь от обычных `messages` к простому агенту.

Что в ней есть:

- переход от большого текстового prompt к `messages`;
- роль `system prompt`;
- объяснение, зачем нужны `tools`;
- хранение простого состояния в `dataclass`;
- проверка данных через `Pydantic`;
- первый базовый агент;
- few-shot для подбора вариаций имени;
- вторая tool для проверки наличия братьев и сестёр;
- итог про то, чем агент отличается от обычного вызова LLM.

На чём акцент:

- понять, как model, prompt, tools и state работают вместе;
- увидеть, как меняется поведение агента при изменении system prompt;
- научиться читать логи шагов агента;
- сравнить, что можно решить промптом, а что лучше решать отдельной tool.

Задания в конце семинара:

- экспериментировать с `SYSTEM_PROMPT_V3` и `SYSTEM_PROMPT_V4`;
- менять сам вопрос и проверять, где агент начинает путаться;
- думать, нужна ли вообще новая tool в конкретной задаче;
- пробовать вопросы, где агент должен выбирать между несколькими tools.
